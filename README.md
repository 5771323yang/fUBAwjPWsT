# 病人信息管理系统 python435

## 项目概述

病人信息管理系统是基于 Python 的 Django 框架开发的，旨在为医疗行业提供功能全面的解决方案。该系统适用于项目实训、在线课程任务等多种场景。

## 技术栈

- 后端：Python 3.x，Django
- 数据库：SQL
- 前端：HTML/CSS/JavaScript

## 功能模块

- 病人信息管理
- 医疗记录管理
- 用户权限管理
- 数据报表

## 系统角色

- 系统管理员
- 医生
- 病人

## 运行环境

- 操作系统：Windows/Linux/MacOS
- Python 环境要求：Python 3.x
- 数据库：兼容主流 SQL 数据库

## 部署步骤

1. 安装 Python 3.x
2. 安装 Django 框架
3. 配置数据库
4. 运行系统
